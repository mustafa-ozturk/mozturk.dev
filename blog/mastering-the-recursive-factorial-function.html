To strengthen my own knowledge, I will try to explain how it works.

Simply put, recursion is a function that calls itself.

Let's say n is 4:

This function will return 4 * factorial(4 - 1).
and to find out what factorial(3) is,

It will call itself with 3 passed in as the argument

factorial(3) will return 3 * factorial(3 - 1)
and to find out what factorial(2) is,

It will call itself with 2 passed in as the argument

factorial(2) will return 2 * factorial(2 - 1)
and to find out what factorial(1) is,

It will call itself with 1 passed in as the argument

now remember the if statement.

Since n is no longer greater than 1 it will simply return 1

This is called the "base case" and it means that there is no recursion left to do (i.e no function of itself left to call).

Now that the function knows factorial(1) is equal to 1
then factorial(2) must equal 2 * 1
and factorial(3) must equal 3 * (2 * 1)
and factorial(4) must equal 4 * (3 * ( 2 * 1) )
and that equals to 24.
